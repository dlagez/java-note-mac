打印缓冲区的数据

```java
byte[] array = byteBuffer.array();
for (int i = 0; i < array.length; i++) {
		if (array[i] == 0) {
		System.out.println("空格");
	} else {
		System.out.println((char) array[i]);
	}
}
```

```java
for (int i = 0; i < byteBuffer.limit(); i++) {
    System.out.println((char) byteBuffer.get());
}
```
