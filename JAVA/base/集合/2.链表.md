链表的每个对象存放单独的链接，每个链接还存放着序列下一个链接的引用。在java中链表都是双向链接的。

```java
public class LinkList_test {
    public static void main(String[] args) {
        LinkedList<String> staff = new LinkedList<>();
        // add方法将对象添加到链表的尾部
        staff.add("roczhang");
        staff.add("dlage");
        staff.add("aaa");
        Iterator<String> iterator = staff.iterator();
        String first = iterator.next();
        String second = iterator.next();
        iterator.remove(); // 这个操作会删除second元素
    }
}
```

如果想要在链表的中间位置插入元素，需要调用器迭代器方法。

比如想要在第一个和第二个元素位置之间插入新的元素。

```java
public class LinkList_test {
    public static void main(String[] args) {
        LinkedList<String> staff = new LinkedList<>();
        // add方法将对象添加到链表的尾部
        staff.add("roczhang");
        staff.add("dlage");
        staff.add("aaa");
        Iterator<String> iterator = staff.iterator();
        String first = iterator.next();
        String second = iterator.next();
        iterator.remove(); // 这个操作会删除second元素
        System.out.println(staff.toString());

        // 调用其listiter方法
        ListIterator<String> iter = staff.listIterator();
        iter.next();
        //  在第一个元素之后插入元素
        iter.add("dlage");
        System.out.println(staff.toString());
    }
}
```

LinkList并不事线程安全的。
