```
sudo npm install npm -g  # 升级npm
npm install npm -g  # win升级npm
```

安装：

```
npm install <Module Name>  # 安装模块
npm install express          # 本地安装
# 将安装包放在 /usr/local 下或者你 node 的安装目录。
npm install express -g   # 全局安装 

# 查看全局安装的包
npm list -g
```

卸载

```
npm uninstall <Module Name>

# c
npm ls
```
